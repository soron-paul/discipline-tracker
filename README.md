# Discipline Tracker  A simple command-line based discipline tracker built with Python.  ## Features - Create a new account with custom disciplines - Track daily progress - Stores data in a local file - Load previous data anytime - Beginner-friendly project  ## How It Works - User chooses to create a new account or load an existing one - Inputs their disciplines (e.g., study, workout, coding) - Each discipline is tracked with a counter - Data is saved in a file for future use  ## Tech Used - Python - File Handling  ## Future Improvements - Add date tracking - Add streak system - Improve UI - Convert to GUI or web app  ## Note This is a beginner project focused on learning file handling and basic program structure in Python.
